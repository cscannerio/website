Features

Support for AWS, Azure, DigitalOcean, Exoscale and GCP

CScanner has built-in support for the following cloud platforms:

  • Amazon Web Services
  • Microsoft Azure
  • DigitalOcean
  • Exoscale
  • Google Cloud Platform

Scans firewall, object storage and tagging configuration

Cscanner scans for policy violations in:

  • Security Group / Firewall configuration in your cloud account
  • Object Storage ACL misconfiguration
  • If cloud resources are properly tagged

Command Line Interface

Cscanner supports scanning cloud accounts by providing a configuraton file and calling it from the command line.

Dashboard

Cscanner Basic and up contains a web dashboard you can use to configure Cscanner, start scans, and view violations in a list.

API

Integrate cscanner with other tools or automate processes using the API. Every function available from the dashboard is also available using the API.

Violation tracking

Track compliance violations on the dashboard and get confirmation when it is fixed.

Ticket support

Get support for your cscanner installation via tickets.

Periodic scanning

Cscanner advanced automatically runs periodic checks on your configured cloud accounts and reports violations automatically.

Keyserver support

Store your cloud credentials in a keyserver instead of cscanner.

SAML login

Connect the cscanner dashboard to your enterprises login system and make sure only authorized personnel can access the dashboard.
